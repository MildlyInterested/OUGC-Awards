OUGC Awards (1.0)

This plugin will allow you to give awards to users and show them in postbit and profile.

Plugin Information:

  • Name: OUGC Awards
  • Description: This plugin will allow you to give awards to users and show them in postbit and profile.
  • Author: Omar Gonzalez
  • Website: uDezain
  • Release Date: April 08, 2012.
  • Version: 1.0
  • MyBB Version: 1.6.7

  • Use external or internal images.
  • Full language support.
  • Send PM when giving awards.
  • Show each award at postbit, profile or both.
  • Deactivate awards without deleting them.
  • Ability to give or revoke awards from the ModCP.

Installation:

  1. Upload the content of the "Upload" folder to your MyBB root folder.
  2. Go to "ACP -> Configuration -> Plugins" and activate the plugin.
  3. Go to "ACP -> Users & Groups -> Manage Awards" to add, give, revoke, find users by award, edit or delete awards.
  4. Activate plugin from "ACP -> Configuration -> OUGC Awards Settings".

Plugin General Settings:

  • Activate Plugin: Turn the award system on/off for outside the ACP.
  • Maximum Awards in postbit: Choose how many awards to show in postbits. Allowed values are:
       "-1" for all
       "number" for the max amount of awards.
       "0" or anything else will disable awards at postbit.
  • Maximum Awards in Profile: Same as above.
  • Show Hidden Awards in ModCP:Allowed moderators to give/revoke hidden awards in the ModCP.
  • Awards moderators:By default, Admins and Super moderators can manage awards from the ModCP (no setting for this), here you can add extra groups that can manage awards as well.

Creating an Award:

  • Name: A short name for the award. (Required)
  • Description: A short description for the award. (Optional)
  • Image: The award image. (Optional)
       "http://..." for external images
       "name.png" for iamges stored in the MYBB_ROOT/images/ougc_awards/ directory if exists.
       "/name.png" for images relative to your forum url if exists.
       Anything else will try to use default image (MYBB_ROOT/images/ougc_awards/default.png).
  • Visible: Whether if the award is active or not at all outside the ACP.
  • PM: PM to be send to users when awarded with this ward. (Optional | Empty = no pm)
       "{1}" = Username
       "{2}" = Award name
       "{3}" = Reason
       "{4}" = Image
  • Type: You can show this award at profile, postbit, or both places.

Giving an Award:

  • Username: Username of the user to receive the award. (Required)
  • Reason: A short reason for the award being given. (Optional)

Revoke an Award:

  • Username: Username of the user to revoke the award from. (Required)

Edit User Award:

  • Award: You can change the award here. (Required)
  • Reason: A short reason for the award being given. (Optional)
  • Time Stamp: The unix timestamp for when the award was given. (Required)

Rebuilt Cache:

  • Cache is suppose to be always up to date, however can always refresh it from the ACP.
